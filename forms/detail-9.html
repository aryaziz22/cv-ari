IoT AquaGrow Project Description

AquaGrow is an IoT-based hydroponic system designed for monitoring and controlling the nutrient and light conditions of kangkung (water spinach) cultivation. The system utilizes ESP8266 NodeMCU, LDR sensors for light intensity, TDS sensors for nutrient concentration, relay modules, nutrient pumps, and UV lamps. Monitoring and control can be performed via a mobile interface, allowing users to access real-time data and control actuators remotely. The system also integrates an ESP32-CAM module for capturing plant images, enabling plant health monitoring using a smart vision system based on decision tree algorithms for image processing and health assessment.

Main Features:
1. Real-time monitoring of light intensity and nutrient concentration
2. Automated control of nutrient pumps and UV lamps
3. Mobile dashboard for data visualization, manual override, and remote access
4. Plant health monitoring using ESP32-CAM and decision tree-based smart vision
5. Alerts for abnormal conditions
6. Data logging for analysis

This project was developed as a semester 5 assignment, focusing on practical IoT and AI applications in smart agriculture.
