Cheerity Campaign Website UI/UX Design Project Description

Cheerity is a UI/UX design project for a charity campaign website, created using Figma. The project includes comprehensive design deliverables such as low-fidelity (lo-fi) wireframes, high-fidelity (hi-fi) mockups, detailed task flows, and a fully interactive prototype. The design aims to provide an engaging, user-friendly experience for users to participate in and support charity campaigns online.

Main Features:
1. Lo-fi and Hi-fi wireframes for all key pages
2. Task flow diagrams for user journeys
3. Interactive prototype demonstrating user experience
4. Modern, accessible, and responsive web design
5. Focus on usability and clear call-to-action for donations and campaign participation
