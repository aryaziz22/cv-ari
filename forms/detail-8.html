Attendance & Timesheet Website Project Description

The Attendance & Timesheet project for CV Alkas Triyasa Adhimukti is a web-based management system designed to simplify and automate employee attendance tracking and timesheet submissions. Built with Laravel, the platform enables employees to check in/out, submit timesheets, and allows HR/admin to monitor and manage attendance records efficiently.

Main Features:
1. Employee check-in/check-out with real-time status
2. Timesheet submission and approval workflow
3. Admin dashboard for attendance and timesheet monitoring
4. Role-based access for employees and HR/admin
5. Responsive design for desktop and mobile

This project was completed as part of an internship program, focusing on practical HR technology solutions for small to medium enterprises.
